Firebase Setup Required

Your admin panel is built — you just need to connect it to Firebase. Follow these steps once.

1
Create a Firebase project Go to console.firebase.google.com, click "Add project", name it chunkz-store, and follow the prompts.
2
Register your web app In your Firebase project → click the </> icon → name it chunkz-web → copy the firebaseConfig object shown.
3
Paste your config into both files Open admin.html and index.html. Find the FIREBASE_CONFIG object in each and replace the placeholder values with your actual keys.
4
Create an admin user In Firebase Console → Authentication → Sign-in method → enable Email/Password. Then go to Users → Add user → enter your email and a strong password. That's your admin login.
5
Enable Firestore & set security rules Firebase Console → Firestore Database → Create database (Start in production mode). Then go to Rules tab and paste:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /orders/{orderId} { allow create: if true; allow get: if true; allow list, update, delete: if request.auth != null; } match /catalog/{docId} { allow read: if true; allow write: if request.auth != null; } match /settings/{docId} { allow read: if true; allow write: if request.auth != null; } } }
6
Deploy & you're live Push your updated files to Vercel. Once deployed, orders will save to Firestore and this admin panel will show your dashboard.
